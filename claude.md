# Claude

Short notes and usage template for working with Anthropic Claude-style assistants.

## Purpose
- Capture prompts, API snippets, and instructions for using Claude.

## Example: Prompt Template
- Goal: Explain a concept clearly and concisely.

```
You are Claude, an assistant that answers concisely.
Explain: <topic>
Audience: <audience>
Length: <short|detailed>
```

## API Notes
- Keep API keys out of repository. Use environment variables like `CLAUDE_API_KEY`.
- Verify rate limits and model versions before production.
